# QEST + FORMATS


## Edit a page

Just edit the corresponding .md file

## Add a page

Add the page.md

Add the page to submenu.yml

## Adding an image

Add the image to the `assets` directory, the image will be at https://confest-2026.github.io/qestformats/assets/imagename.png

## Deploying the page

Simply commit and push to the qestformats repository, this will trigger a rebuild of the main confest website which will automatically incorporate the new qestformats content.
